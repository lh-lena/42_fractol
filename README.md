## Project Information

#### About

Fractol is a graphical project that involves generating and exploring fractals using complex mathematical formulas. The primary objective is to render different types of fractals, such as the Mandelbrot and Julia sets, and provide interactive features to explore them. This project helps in understanding graphical rendering, complex numbers, and user interface interactions.

### Fractals Implemented

##### Mandelbrot Set: 
A set of complex numbers for which the function does not diverge when iterated from ùëß = 0.

##### Julia Set:
Generated by iterating a complex function for each point in the complex plane, using a fixed complex parameter.

### Interactivity and Controls

#### Keyboard and Mouse Events:
Used for zooming, changing fractals, and exiting the program.
For example:
 - **Arrow keys** to move the view
 - **Mouse wheel** to zoom in and out
 - **'c'** to cycle through different color schemes
 - **'p'** to switch to a different Julia set variation
 - **'r'** to reset the view to the default zoom and position
 - **'*'** to increase the maximum number of iterations, enhancing the fractal's detail
 - **'/'** to decrease the maximum number of iterations, simplifying the fractal's detail
 - **'i'** to display the window control options and instructions
 - **ESC** key or click the window's close button to exit the application

### Allowed Functions
In this project, only a restricted set of functions are allowed, ensuring a deep understanding of low-level operations:

- [`malloc`](https://man7.org/linux/man-pages/man3/free.3.html)
- [`free`](https://man7.org/linux/man-pages/man3/free.3.html)
- [`write`](https://man7.org/linux/man-pages/man2/write.2.html)
- [`exit`](https://man7.org/linux/man-pages/man3/exit.3.html)
- [`perror`](https://man7.org/linux/man-pages/man3/perror.3.html)
- [`strerror`](https://man7.org/linux/man-pages/man3/strerror.3.html)
- [`all functions of the math.h`](https://pubs.opengroup.org/onlinepubs/009695399/basedefs/math.h.html)
- [`all functions of the MiniLibX`](https://github.com/42Paris/minilibx-linux)

### Installing and Compiling the Project:
1. Clone the repository:
```C
    git clone git@github.com:lh-lena/42_fractol.git
```
2. Navigate to the project directory:
```C
    cd 42_fractol
```
3. Compile the project: 
```C
    make
```
### Executing the Project
```shell
  ./fractol <fractal type>
```
Available fractals:

	- julia

	- mandelbrot
